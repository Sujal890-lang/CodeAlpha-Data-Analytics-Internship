## 📋 Tasks Overview

### ✅ Task 1: Web Scraping
**Objective:** Extract and create a custom dataset from Amazon Products

**What I did:**
- Collected Amazon product data using Kaggle dataset
- Cleaned prices (removed ₹ symbols, commas)
- Cleaned ratings, categories, and review counts
- Removed duplicates and missing values
- Added new features: `Price_Savings` and `Rating_Label`
- Saved a clean custom dataset for further analysis

**Tools & Libraries:**
`Python` `Pandas` `BeautifulSoup4` `Requests` `NumPy`

**Output:** `amazon_cleaned_dataset.csv`
